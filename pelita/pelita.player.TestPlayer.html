pelita.player.TestPlayer¶

class pelita.player.TestPlayer(moves)[source]¶

Bases: pelita.player.AbstractPlayer

A Player with predetermined set of moves.

Parameters:

moves : list of moves or str of shorthand symbols

the moves to make in order, see notes below

Notes

The moves argument can either be a list of moves, e.g. [west, east, south, north, stop] or a string of shorthand symbols, where the equivalent of the previous example is: '><v^-'.
