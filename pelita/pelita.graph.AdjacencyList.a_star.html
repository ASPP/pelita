pelita.graph.AdjacencyList.a_star¶

AdjacencyList.a_star(initial, target)[source]¶

A* search.

A* (A Star) [1] from one position to another. The search will return the shortest path from the initial position to the target using the Manhattan distance as a heuristic.

Parameters:

initial : tuple of (int, int)

the first position

target : tuple of (int, int)

the target position
Returns:

path : lits of tuple of (int, int)

the path from initial to the closest target
Raises:

NoPathException :

if no path from initial to one of targets

NoPositionException :

if either initial or targets does not exist

[1] http://en.wikipedia.org/wiki/A*_search_algorithm :
