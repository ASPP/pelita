pelita.player¶

Base classes for player implementations.

Classes

AbstractPlayer Base class for all user implemented Players.
CPUBoundPlayer Player that does loads of computation.
DebuggablePlayer Player which invokes pdb on each move.
IOBoundPlayer IO Bound player that crawls the file system.
InitialExceptionPlayer Player that raises an Exception on set_initial().
MoveExceptionPlayer Player that raises an Exception on get_move().
RoundBasedPlayer(moves) A Player which makes a decision dependent on the round index in a dict or list.
SimpleTeam(*args) Simple class used to register an arbitrary number of (Abstract-)Players.
SpeakingPlayer A player that makes moves at random and tells us about it.
StoppingPlayer A Player that just stands still.
TestPlayer(moves) A Player with predetermined set of moves.
