Adjacency list [1] representation of a Maze.

The AdjacencyList is mostly a wrapper for a dict. Given a position, it returns the positions reachable from there.

a_star(initial, target)[source]¶

A* search.

A* (A Star) [1] from one position to another. The search will return the shortest path from the initial position to the target using the Manhattan distance as a heuristic.

Parameters:
  • initial (tuple of (int, int)) – the first position
  • target (tuple of (int, int)) – the target position
Returns:

path – the path from initial to the closest target
Return type:

lits of tuple of (int, int)
Raises:
  • NoPathException – if no path from initial to one of targets
  • NoPositionException – if either initial or targets does not exist
  • [1] http://en.wikipedia.org/wiki/A*_search_algorithm
