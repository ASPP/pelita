The Graph is mostly a wrapper for a dict. Given a position, it returns the positions reachable from there.

a_star(initial, target)[source]¶

A* search.

A* (A Star) [1] from one position to another. The search will return the shortest path from the initial position to the target using the Manhattan distance as a heuristic. Algorithm here is partially taken from [2] and inlined for speed.

Parameters

  • initial ((int, int)) – the initial position

  • target ((int, int)) – the target position

Returns

path – one of the the shortest paths from initial to the closest target (excluding the initial position itself)

Return type

list of (int, int)

Raises

  • NoPathException – if no path from initial to one of targets

  • [1] http://en.wikipedia.org/wiki/A*_search_algorithm –

  • [2] http://www.redblobgames.com/pathfinding/a-star/implementation.html#python –

bfs(initial, targets)[source]¶

Breadth first search (bfs).

Breadth first search [1] from one position to multiple targets. The search will return a path from the initial position to the closest position in targets.

Parameters

  • initial (tuple of (int, int)) – the first position

  • targets (list of tuple of (int, int)) – the target positions

Returns

path – the path from initial to the closest target

Return type

lits of tuple of (int, int)

Raises

  • NoPathException – if no path from initial to one of targets

  • NoPositionException – if either initial or targets does not exist

pos_within(position, distance)[source]¶

Positions within a certain distance.

Calculates all positions within a certain distance of a target position in maze space. Within means strictly less than (<) in this case.

Parameters

  • distance (int) – the distance in maze space

Returns

pos_within – the positions within the given distance

Return type

set of tuple of (int, int)

Raises

NoPathException – if position does not exist in the adjacency list
