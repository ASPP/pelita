pelita.messaging.DispatchingActor¶

class pelita.messaging.DispatchingActor(**kwargs)¶

Bases: pelita.messaging.actor.Actor

The DispatchingActor allows methods of the form

@expose def some_action(self, *args)

which may be called as

actor = DispatchingActor() actor.send(“some_action”, params)

An alternative form which allows for calling with a different name is available

@expose(name=”action”) def some_action(self, *args)

actor.send(“action”, params)

Note that DispatchingActor overrides on_receive.
