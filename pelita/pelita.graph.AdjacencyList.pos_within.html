pelita.graph.AdjacencyList.pos_within¶

AdjacencyList.pos_within(position, distance)¶

Position within a certain distance.

Calculates all positions within a certain distance of a target position in maze space. Within means strictly less than (<) in this case.

Parameters :

position : tuple of (int, int)

the first position

distance : int

the distance in maze space
Returns :

pos_within : set of tuple of (int, int)

the positions within the given distance
Raises :

NoPositionException :

if either initial or targets does not exist
