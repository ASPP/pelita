pelita.datamodel.MazeComponent¶

class pelita.datamodel.MazeComponent¶

Bases: object

Base class for all items inside a Maze.

This class provides basic methods for serialisation but is not serialisable itself (it does not have a _json_id). This is to ensure that all inherited objects are decorated with @serializable and do not falsely inherit the id from this class.
