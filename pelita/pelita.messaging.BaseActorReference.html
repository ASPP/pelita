pelita.messaging.BaseActorReference¶

class pelita.messaging.BaseActorReference(**kwargs)¶

Bases: pelita.messaging.actor.Channel

An ActorReference is used to send all requests and notifications to the actor. It also holds the currently processed message and information about the sender.

Every local ActorReference has a 1:1 reference to an Actor. The splitting is due to the fact that the Actor class must be subclassed and thus we avoid some name clashes.

Helper class to send messages to an actor.
