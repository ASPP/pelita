pelita.messaging.ActorReference.join¶

ActorReference.join(timeout=None)¶

Blocks until the actor’s thread is completed or waits timeout seconds. Whatever happens earlier.

Parameters :

timeout : float, optional

the time in seconds to wait. default = None (no timeout)
