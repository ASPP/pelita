pelita.simplesetup.RemoteTeamPlayer¶

class pelita.simplesetup.RemoteTeamPlayer(socket)[source]¶

Bases: object

This class is registered server-side with the GameMaster and sends all requests to the attached zmq socket (to which a client player has connected.)

It also does some basic checks for correct return values.

Parameters:

socket : zmq socket

The zmq socket of this connection
