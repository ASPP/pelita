pelita.layout.get_available_layouts¶

pelita.layout.get_available_layouts(filter='')¶

The names of the available layouts.

Parameters :

filter : str

only return layouts which contain ‘filter’ in their name. Default is no filter.
Returns :

layout_names : list of str

the available layouts

Examples

To only get layouts without dead ends you may use:

>>> get_available_layouts(filter='without_dead_ends')
