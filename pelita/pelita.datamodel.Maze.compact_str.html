pelita.datamodel.Maze.compact_str¶

Maze.compact_str¶

Return a compact string representation of the Mesh.

This is useful in case the maze contains components that can be represented with single character strings. See the following examples for details.

Non-compact string:

['#', '#', '#', '#', '#', '#']
['#', ' ', ' ', '#', '0', '#']
['#', ' ', '3', ' ', '#', '#']
['#', '2', ' ', ' ', '1', '#']
['#', '#', '#', '#', '#', '#']

Compact string:

######
#  #0#
# 3 ##
#2  1#
######
Returns:

compact : str

compact string representation
