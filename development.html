Development¶

This section contains information for developers of Pelita.

Git-Repository¶

The official Git-Repository is hosted at Github: https://github.com/Debilski/pelita

You can create a clone with:

$ git clone git://github.com/Debilski/pelita.git

Project-Wall¶

We used an agile project wall to track tasks for the initial stages of the development. In our case we split it into four distinct sections: todo (upper right), current iteration (upper left), done (lower left) and nice to have (lower right). Since we only had one whiteboard we kept the center for sketches and doodles. You can read about the agile project wall in the book: The Agile Samurai by by Jonathan Rasmusson
